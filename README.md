# goNode

### It's a simple nodejs start up to try experimenting with some settings

## Project start up

```terminal
yarn install    # install npm packages
yarn start      # run application
```

## Troubleshooting

To solve EADDRINUSE

```terminal
sudo lsof -i :3000
kill -9 <PID>
```
